## Project Description

VoxelForge is a desktop application that uses generative AI to transform natural language prompts into Minecraft-compatible 3D structures. Users can describe a structure in plain text, and the software generates a 3D model, converts it into a voxel representation, and exports it as a structure file that can be imported directly into Minecraft building tools.

The goal of VoxelForge is to make advanced structure creation accessible to everyone, allowing players and builders to generate complex designs without needing 3D modelling experience. By combining AI-generated meshes with a configurable voxelisation pipeline and Minecraft block palette system, the application bridges the gap between modern AI generation and the Minecraft building ecosystem.

VoxelForge supports multiple export formats commonly used by the Minecraft community, including `.schematic`, `.schem`, and `.litematica`. Users can control generation resolution, restrict block palettes to survival-obtainable blocks or full cubes, preview the voxelised structure before export, and customise output settings through a simple desktop interface.

The project focuses on providing a clean and intuitive workflow: prompt → AI generation → voxelisation → Minecraft structure export. The result is a fast and user-friendly tool for turning creative ideas into buildable Minecraft structures within seconds.
